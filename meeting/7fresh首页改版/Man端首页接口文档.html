Type: GET

Content-Type: application/json

Request-example:

No request parameters are required.

Response-fields:

Field Type Description
success boolean 是否成功
code string 响应代码
message string 响应信息
data object 响应数据
└─pluginCode string 插件编码
└─name string 插件名称
└─remark string 插件备注
└─contentUrl string 插件内容url

└─contentStyle string 插件内容样式
└─seq int 插件排序
└─config string 插件配置
